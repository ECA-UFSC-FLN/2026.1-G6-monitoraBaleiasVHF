# 📌 Monitoramento de Baleias por VHF

O projeto desenvolve um sistema digital para auxiliar na localização dos tags VHF utilizados pelo Projeto MARESIS no monitoramento de baleias-jubarte. Esses tags armazenam dados científicos e, após se desprenderem do animal, precisam ser recuperados em campo. O método atual, baseado apenas em detecção auditiva do sinal, é lento, impreciso e depende da experiência do operador.
A solução proposta consiste em um protótipo de hardware e software capaz de estimar a direção de origem do sinal VHF por meio de um arranjo de múltiplas antenas e receptores RTL-SDR, utilizando técnicas de Direction of Arrival (DOA). O sistema exibe a direção estimada em uma interface gráfica visual e intuitiva, permitindo buscas mais rápidas, eficientes e confiáveis. O objetivo principal é aumentar a taxa de recuperação dos tags, garantindo que os dados científicos coletados sejam preservados e facilitando o trabalho das equipes de campo.
